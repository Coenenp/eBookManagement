Configuration Tips
  • Optional Setup: All scrapers are optional. The library works great with just file-based metadata.
  • API Keys: You can add API keys later in the settings if you skip this step.
  • Privacy: API keys are stored securely and only used for metadata requests.
  • Rate Limits: Most services have daily request limits, but they're usually sufficient for personal use.
  • Offline Mode: The library works perfectly without internet connectivity using built-in metadata extraction.
Performance Benefits

Configured scrapers will automatically enrich your library with professional metadata, high-quality covers, and detailed information. This happens in the background and makes browsing and organizing your collection much more enjoyable.
