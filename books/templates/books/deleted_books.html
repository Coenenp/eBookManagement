How to Manage Deleted Books
To Restore a Book:
  1. Ensure the book's file is back on disk (restore from backup)
  2. Run a rescan on the folder containing the file
  3. The scanner will automatically detect and reactivate the book
  4. OR use the Admin interface to manually restore (click Edit → uncheck "deleted_at")
To Permanently Delete:
  1. Go to the Admin Book List
  2. Filter for deleted books (already filtered in link)
  3. Select books to permanently delete
  4. Choose "⚠️ Permanently delete selected books" from the Actions dropdown
  5. Warning: This cannot be undone - all metadata will be lost
Bulk Restore:
  1. In the Admin, select deleted books
  2. Choose "♻️ Restore soft-deleted books" from Actions
  3. This will clear the deleted_at timestamp and make books active again
